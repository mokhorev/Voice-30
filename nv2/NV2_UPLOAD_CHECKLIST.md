# NV2_UPLOAD_CHECKLIST.md

## Как заменить старый комплект

1. Старые файлы без `NV2_` перенести в `old_versions`.
2. В Project Instructions вставить только `NV2_PROJECT_INSTRUCTIONS_SHORT.md`.
3. Остальные файлы v2.1 загрузить как Project Files.
4. Проверить командой:

```text
/фокус проверь файлы проекта и скажи, есть ли конфликтующие инструкции
```
