# НейроВижин v2.1 — sprint pack

Версия: 2026-07-03.

Пакет заменяет старый canonical pack и добавляет коммерческую модель доставки:

```text
тизер → платный аудит → /before → SPRINT_REPAIR 4 недели → проверка ремонта → повторный замер → мониторинг
```

## Как вставить

1. В Project Instructions вставить только `NV2_PROJECT_INSTRUCTIONS_SHORT.md`.
2. В Project Files / Google Drive загрузить весь комплект v2.1.
3. Старые файлы без `NV2_` и старый canonical pack перенести в `old_versions`.
4. Проверить командой:

```text
/фокус проверь файлы проекта и скажи, есть ли конфликтующие инструкции
```

## Главное в v2.1

- `SPRINT_REPAIR` — 4-недельный спринт починки источников.
- `BEFORE_EVIDENCE_PACK` — обязательная фиксация «до».
- `CLAIM_GATE` — сильные продажи без вранья.
- `NV2_SHORT_ANSWERS.md` — краткие ответы для продаж.
- Нишевые адаптеры: industrial product, local service, education, medical device.

## Запреты

Не обещаем место в ChatGPT, Алисе, Яндексе, 2ГИС, Google, картах или «топ ответа». Не используем фейковые отзывы, накрутки, ботов, спам, обход капчи, агрессивный скрейпинг. Не раскрываем полный ремонт в бесплатном тизере.
